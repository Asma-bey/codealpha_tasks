# 🎓 Student Study Assistant Chatbot

An interactive FAQ-based chatbot built using **NLP techniques** and **Streamlit**, designed to help students with study-related questions and provide motivational support.

---

##  Features

-  Ask study-related questions
-  NLP-based FAQ matching (TF-IDF + Cosine Similarity)
-  Preprocessing using NLTK
-  Motivational quotes generator
-  Simple and interactive web UI using Streamlit

---

##  Tech Stack

- Python
- Streamlit
- NLTK
- Scikit-learn
